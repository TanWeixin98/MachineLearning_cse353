README
Required Java Version: 1.8x 
Required Jar: Jama-1.0.3
Required Language Level(project setting for Intellij): 8

Getting Started
Please add the Jama-1.0.3 jar into the project setting. 

Ex. If you are using Intelliji IDEA, go to File->Project Structure-> Libraries (left side of the popup) and click the “+” to select the Jama-1.0.3.jar file. Then click apply. 
Note: The Jama-1.0.3.jar is located inside the External Library folder

The Jama-1.0.3.jar is located inside the External Library folder of the root directory.
